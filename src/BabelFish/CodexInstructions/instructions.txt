The goal is to implement the API call in APIClients/OrionMatchAPIClient for the lambda function in CodexInstructions/lambda_function.py

Perform the following steps:
1. Create the data struction of the API response in DataModel under the correct subdir. 
- In this case would be TournamentSearchList

2. Create the response wrapper for the response data structure in Responses/OrionMatchAPI 
- In this case would wrap TournamentSearchList

3. Create Requests/OrionMatchAPI/TournamentSearchAuthenticatedRequest.cs following the convention of other Authenticated requests in that folder
When writing the Request, make sure to set the correct SubDomain (AUTHAPI versus API). 
- In this case because the response includes continuation tokens, this must be a ITokenRequest. Look at other requests that use ITokenRequest to see how to use it. 
- Do not overload the request construtor, just do one construtor with only UserAuthentication credentials. 

4. Create Responses/OrionMatchAPI/TournamentSearchAuthenticatedResponse.cs following the convention of other Authenticated responses in that folder
- In this case becuase the response includes continuation tokens, this must be a ITokenResponse. Look at other responses that use ITokenResponse to see how to use it. 

Note for the Requests/Responses, do NOT create abstract Response/Request classes

5. Add TournamentSearchAuthenticatedAsync function(s) to APIClients/OrionMatchAPIClient.cs following the convention of other API call functions in that file

6. Create/write new test functions to test the function created in OrionMatchAPIClient.cs in tests/BabelFish.Tests/OrionMatch/Tournament/GetTournamentTests.cs 

DO NOT TRY TO RUN ANY .NET COMMANDS such as building. 

If an enum for a request parameter already exists, use it and use EnumHelper.MemberValue to convert to query string (not applicable in this case)
